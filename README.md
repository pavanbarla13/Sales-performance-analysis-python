# Sales Performance Analysis (Python)

## Project Objective
Analyze sales data to identify:
- Monthly revenue trends
- Top performing cities
- Best selling products
- Highest revenue categories

## Tools Used
- Python
- Pandas
- Git & GitHub

## Key Analysis Performed
1. Converted order_date to datetime format
2. Created total_amount column (quantity * price)
3. Monthly revenue aggregation
4. City-wise revenue ranking
5. Product-wise revenue ranking
6. Category-wise revenue ranking

## How to Run
1. Install pandas
2. Place sales_raw.csv in the same folder
3. Run: python sales_performance_analysis.py
